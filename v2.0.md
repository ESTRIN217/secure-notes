# Secure Notes - Release v2.0.0

---

## 🇺🇸 English

### 🔄 Interactive Drag & Drop in Custom Sort Mode
We have introduced a fully intuitive and interactive drag-and-drop feature to reorder your notes dynamically in the **Grid View**.
* **Lifting Feedback:** Long-pressing a note card will visually "lift" it (scaling by 6% and updating depth elevation).
* **Smooth Swapping:** Dragging a note over other columns or rows dynamically swaps positions instantly, animating beautifully in real time.
* **Persistent Position:** The custom order is stored securely in local preferences, ensuring your custom layout remains exactly as you left it.

### 🎨 Design Polish & Bugfixes
* Implemented Material Design 3 Expressive Outlined border cards.
* Cleaned up UI padding and enhanced screen contrast for better readability in both light and dark modes.

### 🖌️ Floating Toolbar, Note Styling, and Rich Sharing Options (v2.0.0)
* **Floating Style Palette:** A new floating toolbar at the bottom of the editor houses a palette icon, opening a customization sheet to pick note colors and select custom background images from device storage.
* **More Actions Sheet:** Replaced individual app bar buttons with an elegant "More Options" bottom sheet containing native note sharing options (TXT, Markdown, PDF, HTML, JSON) and immediate note deletion.
* **Last Modified Badge:** Displays a prominent, styled badge detailing when the note was last edited above the content area.
* **Quick Note Status Controls:** Added dedicated, interactive action icons in the editor's app bar to instantly pin notes, add them to favorites, and archive them directly without leaving the editing workspace.
* **Seamless Auto-Save on Exit:** Removed the manual save button in favor of an instant auto-save mechanism. The editor now automatically saves your draft whenever you navigate back or exit the editor screen, with a clean status confirmation Toast.
* **Multimodal Toolbar Features:**
  * **Text-to-Speech (TTS):** Read note titles and contents aloud instantly with a custom play/stop toolbar controller.
  * **Handwritten Drawings:** A dedicated immersive drawing canvas screen with custom stroke width controls, rich color selection palette, and real-time canvas rendering. Saves drawings directly as high-resolution PNG files linked to your note.
  * **Comprehensive Attachment Hub:** A new Modal Bottom Sheet dialog offering options to record live high-quality voice notes (with real-time microphone recording and playbacks) and attach external system files natively, copied securely to internal storage.

### ✍️ WYSIWYG Advanced Rich Text Editor & Formatter
A premium WYSIWYG (What You See Is What You Get) rich text editing suite has been fully integrated into the note-creation space, complete with complete localized support:
* **Interactive Style Insertion:** Select any block of text to instantly apply Bold, Italic, Underline, and Strikethrough style tags directly.
* **Inline Code Formatting:** Support for raw monospace `<code>` formatting block wraps to capture technical drafts and code blocks.
* **Subscript & Superscript:** Easily apply structural layout styles like subscript (`<sub>`) and superscript (`<sup>`).
* **Font Color & Background Grid Picker:** Apply premium, high-contrast colored styling highlights (red, blue, green, yellow, orange, purple, pink, brown, black, gray) using inline tag mappings (`<color>` and `<bg>`).
* **Heading Style Selection:** Structure notes cleanly using dedicated Heading 1, Heading 2, and Heading 3 format headings.
* **Clear Formatting Utility:** One-tap action button to strip all tags or styled formats from the selected text block instantly.
* **Inline Media Attachments:** Dynamically insert external or local image (`<img>`) and video (`<video>`) source links right into the body text, rendering them as beautiful responsive visual elements inside the reader mode layout.
* **Structured List Styling:** Native formatting support for Numbered Lists (`<ol><li>`), Bulleted Lists (`<ul><li>`), and interactive checkable Checklists (`<cl><item checked="false">`).
* **Code Blocks & Blockquotes:** Wrap structural elements in monospace block `<pre>` tags for beautiful code blocks and italicized muted `<quote>` blocks.
* **Indentation Navigation:** Seamlessly increase or decrease block indentation level wraps (`<indent>`) dynamically.
* **URL Hyperlinks:** Insert formatted high-contrast hyperlinks (`<url=address>`) with customizable text wrappers.
* **Interactive Text Search:** Live search utility in the editing workspace to query and navigate occurrences (Previous/Next matches).
* **Clipboard & Temporal Tools:** Instantly paste clipboard content at cursor position, and insert beautiful localized current date/time stamps.
* **Undo & Redo System:** Seamlessly traverse your drafting history using the intuitive Undo and Redo control buttons with debounced, character-safe history saving.
* **Premium Font Families:** Customize your notes using a typography selector supporting Serif, Monospace, Sans-Serif, and Cursive fonts.
* **Dynamic Font Scaling:** Scale text to any specific font sizing (from 12sp up to 28sp) for maximum layout emphasis and styling.
* **True WYSIWYG Real-Time Editor:** Write and edit directly in a single input field that dynamically parses and hides Markdown/HTML tags, showing styled rich text instantly with perfect cursor alignment.

## 🇻🇪 Español (VE)

### 🔄 Drag & Drop Interactivo en Modo de Orden Personalizado
Hemos introducido una función de arrastrar y soltar (drag-and-drop) totalmente intuitiva e interactiva para reordenar tus notas de manera dinámica en la **Vista de Rejilla**.
* **Retroalimentación de Elevación:** Al mantener presionada una tarjeta de nota, esta se "levantará" visualmente (aumentando su escala un 6% y actualizando su nivel de profundidad).
* **Intercambio Fluido:** Arrastrar una nota sobre otras columnas o filas intercambia su posición de forma instantánea, animándose maravillosamente en tiempo real.
* **Persistencia de Posición:** El orden personalizado se almacena de forma segura en las preferencias locales, asegurando que tu diseño permanezca exactamente como lo dejaste.

### 🎨 Pulido de Diseño y Corrección de Errores
* Implementación de tarjetas con bordes perfilados (*outlined*) de Material Design 3 Expresivo.
* Optimización de márgenes y espaciados de la interfaz para una mejor legibilidad en temas claro y oscuro.

### 🖌️ Barra Flotante, Estilo de Notas y Opciones de Compartido Enriquecido (v2.0.0)
* **Paleta de Estilo Flotante:** Una nueva barra de herramientas flotante en la parte inferior del editor alberga un ícono de paleta, abriendo un panel para personalizar colores de notas y seleccionar imágenes de fondo desde el almacenamiento del dispositivo.
* **Panel de Más Acciones:** Reemplazamos los botones individuales de la barra superior con un elegante panel de "Mais Opciones" que ofrece opciones nativas para compartir notas (TXT, Markdown, PDF, HTML, JSON) y eliminación rápida.
* **Etiqueta de Última Modificación:** Muestra un distintivo estilizado arriba del contenido detallando la última fecha y hora de edición.
* **Controles Rápidos del Estado de Notas:** Se añadieron íconos de acción dedicados e interactivos en la barra superior del editor para fijar notas, agregarlas a favoritos y archivarlas directamente sin salir del área de edición.
* **Auto-guardado automático al salir:** Eliminamos el botón manual de guardar a favor de un mecanismo de guardado automático instantáneo. El editor ahora guarda automáticamente tu borrador al retroceder o salir de la pantalla de edición, mostrando una notificación Toast para confirmar que la nota ha sido guardada de forma segura.
* **Funciones de Barra Multimodal:**
  * **Texto a Voz (TTS):** Escucha el título y contenido de tu nota en voz alta al instante con un controlador interactivo en la barra de herramientas.
  * **Dibujos a Mano Alzada:** Una pantalla de lienzo dedicada e inmersiva para dibujar con controles de grosor de trazo, paleta de selección de colores y guardado directo de imágenes PNG de alta resolución asociadas a tu nota.
  * **Centro Completo de Adjuntos:** Un nuevo panel inferior interactivo (*Modal Bottom Sheet*) que permite grabar notas de voz en tiempo real con controles de preescucha, y adjuntar archivos externos de almacenamiento copiados localmente de forma segura.

### ✍️ Editor de Texto Enriquecido Avanzado WYSIWYG
Se ha integrado una suite premium de edición de texto enriquecido WYSIWYG (What You See Is What You Get - Lo que ves es lo que obtienes) en el espacio de creación de notas, con soporte nativo de traducción:
* **Inserción de Estilo Interactiva:** Selecciona cualquier bloque de texto para aplicar instantáneamente etiquetas de Negrita, Cursiva, Subrayado y Tachado.
* **Formato de Código Inline:** Soporte para bloques monoespaciados `<code>` para capturar fragmentos técnicos o código de programación.
* **Subíndice y Superíndice:** Aplica estilos estructurales como subíndice (`<sub>`) y superíndice (`<sup>`).
* **Selector de Color de Fuente y Fondo:** Aplica resaltados de color premium de alto contraste (rojo, azul, verde, amarillo, naranja, púrpura, rosa, marrón, negro, gris) usando etiquetas integradas (`<color>` y `<bg>`).
* **Estilos de Encabezado:** Estructura notas limpiamente utilizando títulos dedicados de Encabezado 1, Encabezado 2 y Encabezado 3.
* **Herramienta para Limpiar Formato:** Botón de acción rápida para eliminar instantáneamente todas las etiquetas o estilos de formato del bloque de texto seleccionado.
* **Inserción de Múltiples Medios:** Inserta dinámicamente enlaces a imágenes (`<img>`) y videos (`<video>`) en el texto, representándolos como elementos visuales responsivos dentro de la vista previa.
* **Estructuras de Lista Avanzadas:** Soporte de formato nativo para Listas Numeradas (`<ol><li>`), Listas con Viñetas (`<ul><li>`), y Listas de Tareas interactivas completables (`<cl><item checked="false">`).
* **Bloques de Código y Citas:** Organiza párrafos con bloques de código monoespaciados `<pre>` y bloques de citas estilizados en cursiva y tono atenuado `<quote>`.
* **Ajuste de Sangrías Dinámico:** Aumenta o disminuye fácilmente los niveles de sangrado y estructuración del bloque de texto (`<indent>`).
* **Hipervínculos URL:** Inserta enlaces web de alto contraste (`<url=dirección>`) con texto personalizado.
* **Buscador de Texto Interactivo:** Herramienta de búsqueda en tiempo real dentro del editor de notas para localizar y navegar coincidencias (Siguiente/Anterior).
* **Herramientas de Portapapeles y Fecha:** Pega al instante contenido del portapapeles en el cursor e inserta marcas de fecha y hora del sistema con un toque.
* **Sistema de Deshacer y Rehacer:** Navega sin problemas por tu historial de borrador utilizando los intuitivos botones de control Deshacer y Rehacer con guardado de historial seguro y diferido.
* **Fuentes Tipográficas Premium:** Personaliza tus notas con un selector tipográfico que admite fuentes Serif, Monospace, Sans-Serif y Cursive.
* **Escala de Fuentes Dinámica:** Ajusta el texto a tamaños de fuente específicos (desde 12sp hasta 28sp) para lograr el máximo énfasis visual y diseño.
* **Editor WYSIWYG en Tiempo Real:** Escribe y edita directamente en un único campo de entrada que analiza y oculta dinámicamente las etiquetas Markdown/HTML, mostrando texto enriquecido con estilo al instante y con una alineación perfecta del cursor.
