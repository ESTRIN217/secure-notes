# v3.0 — Settings Overhaul

## Nuevas pantallas dedicadas

- **Settings Hub** — Pantalla central de configuración con secciones organizadas
- **Privacy & Security** — Configuración de contraseña maestra (crear/eliminar)
- **Backup & Restore** — Copias de seguridad local (SAF) + nube (Google Drive)
- **Update Info** — Versión actual, ajustes de actualización, buscar nueva versión (GitHub API)
- **About** — Información de la aplicación

## Widgets MD3 Expressive Compartidos

- **SettingsSectionTitle** — Título de sección con tipografía ExtraBold, padding 24dp
- **SettingsIconContainer** — Contenedor de icono 44dp con esquinas 16dp, colores de estado
- **SettingsCardGroup** — Tarjeta MD3 con esquinas 28dp, borde 1.5dp, superficie elevada
- **SettingsSwitchTile** — Fila completa clicable con subtítulo y switch MD3
- **SettingsListTile** — Fila con icono, subtítulo y chevron de navegación

## Apariencia

- **Colores dinámicos**: Switch para activar/desactivar (Android 12+)
- **Modo oscuro**: Ahora tri-estado — Sistema (predeterminado), Apagado, Encendido

## Idioma

- Selección de idioma con opciones: Sistema (predeterminado), English, Español (Venezuela), Português (Brasil)
- Cambio instantáneo vía `Activity.recreate()`

## Privacidad y Seguridad

- **Pantalla dedicada** para configurar la contraseña maestra
- **Crear contraseña** con confirmación y validación de seguridad
- **Eliminar protección** con verificación de contraseña actual
- **Indicador visual** de estado de seguridad activo/inactivo

## Almacenamiento y datos

- **Backup local**: Exportar/Importar todas las notas + tags como JSON vía Storage Access Framework
- **Backup nube**: Google Drive (sync/restore existente movido a esta pantalla)

## Información

- **Registro de cambios**: Bottom sheet con últimas release notes desde GitHub
- **Buscar actualización**: Consulta API de GitHub releases y compara versiones
- **Sobre la aplicación**: Pantalla con icono, versión, descripción y enlaces

## Versión

- `versionName = "3.0"`
- `versionCode = 2`

## Técnico

- Nuevo paquete: `com.example.ui.settings`
- **ViewModels separados**: `ThemeViewModel` (tema/idioma), `BackupViewModel` (respaldos), `UpdaterViewModel` (actualizaciones)
- `DarkModeOption` enum: `SYSTEM / OFF / ON`
- Nuevas keys en `SharedPreferences`: `dynamic_colors`, `dark_mode_option`, `language`, `auto_update_check`, `update_notifications`
- `dynamicColor` ahora configurable (antes hardcodeado `true`)
- Dependencias: OkHttp (ya existente) reutilizado para GitHub API
